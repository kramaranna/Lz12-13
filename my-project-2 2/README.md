# Опис завдання:
Потрібно створити текстовий файл 2D.txt у програмі Блокнот, записати в нього матрицю розміром 3×3. Програма має зчитати цю матрицю з файлу, збільшити всі парні елементи на 10, вивести початкову та змінену матриці на екран, а також зберегти результуючу матрицю у новий файл result.txt.

# Алгоритм виконання:

- Створити у Блокноті файл 2D.txt і ввести в нього 9 чисел у вигляді матриці.

- Відкрити файл 2D.txt для читання за допомогою ifstream.

- Зчитати всі елементи у двовимірний масив matrix[rows][cols].

- Вивести на екран початкову матрицю.

- Пройти по всіх елементах масиву (якщо число парне (% 2 == 0), збільшити його на 10).

- Вивести оновлену (результуючу) матрицю на екран.

- Відкрити новий файл result.txt для запису за допомогою ofstream.

- Записати результуючу матрицю у файл.

- Закрити обидва файли після завершення роботи.
