# Опис логіки програми:

Програма оголошує структуру Student з полями: ПІБ, номер групи та рік народження.

- Створюється масив з 5 студентів для збереження введених даних.

- Користувач вводить ПІБ, номер групи та рік народження для кожного студента.

- Масив студентів сортується за номером групи методом бульбашки.

- Відсортований список студентів виводиться на екран і записується у файл results.txt.

- Користувач вводить рік народження для пошуку.

- Програма знаходить студентів, народжених у цьому році, та виводить їх на екран і записує у файл.

- Якщо студентів з таким роком народження немає, програма виводить відповідне повідомлення.
- Файл results.txt закривається після запису даних.
