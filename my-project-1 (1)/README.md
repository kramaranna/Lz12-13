# Опис завдання:
Необхідно зчитати з файлу number.txt послідовність чисел і знайти добуток усіх чисел, що діляться на 4. Якщо таких чисел немає, вивести відповідне повідомлення.

## Алгоритм виконання:

- Відкрити файл number.txt для читання за допомогою ifstream.

- Перевірити, чи файл успішно відкрився.

- Ініціалізувати змінну count = 1.

- Послідовно зчитувати числа з файлу.

- Для кожного числа перевіряти, чи ділиться воно на 4 (n % 4 == 0).

- Якщо ділиться — перемножити його з попереднім добутком (product *= n).
- Закрити файл.
